# Computer_Vision---Classification_of_Histopathological_Images
Este projeto implementa um pipeline completo de Visão Computacional para a classificação de imagens de imuno-histoquímica de câncer de mama em quatro categorias (0, 1, 2 e 3). O objetivo central é comparar métodos tradicionais de extração de características com modelos de Deep Learning modernos utilizando Transfer Learning.
